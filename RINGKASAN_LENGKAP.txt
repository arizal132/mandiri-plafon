## ğŸ› ï¸ Customization Guide

### 1. Mengubah Nomor WhatsApp
```html
Cari: 6285773907878
Ganti dengan nomor Anda (format: 62xxxxx)
Lokasi: Banyak tempat di index.html
```

### 2. Mengubah Warna Theme
```html
Di <head>, cari section <style>:
.gradient-primary {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

Ganti dengan warna Anda
```

### 3. Mengubah Teks Konten
```html
Buka index.html
Edit teks langsung
Contoh:
<h1>Judul</h1> â†’ <h1>Judul Baru Anda</h1>
```

### 4. Menambah Foto Portfolio
```
1. Simpan foto ke: assets/images/
2. Buka index.html, cari portfolio section
3. Ubah dari:
   <i class="fas fa-image"></i>
   Menjadi:
   <img src="assets/images/nama-foto.jpg" alt="Deskripsi">
4. Update CSS di assets/css/custom.css:
   .portfolio-image img {
       width: 100%;
       height: 100%;
       object-fit: cover;
   }
```

### 5. Mengubah Font
```html
Di <head>, cari:
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;500;600;700;800&family=Playfair+Display:wght@700&display=swap">

Ganti dengan font dari: https://fonts.google.com
```
